## 📅 March 15, 2026

### 🧩 Lab Title:
**Develop an App with Gemini Assistance**

🔍 Objective:  
Menggunakan Gemini Code Assist untuk membangun dan men-deploy aplikasi Flask (Python) ke Cloud Run secara cepat dan efisien.

🔧 Tools/Services Used:
- **Gemini Code Assist**
- **Cloud Workstations**
- **Cloud Run**
- **Python (Flask)**

🧠 What I Did:
- Setup environment development menggunakan **Cloud Workstations**.
- Bertanya ke Gemini tentang perbedaan **Cloud Run** dan **Cloud Functions** untuk menentukan arsitektur terbaik.
- Membuat aplikasi "Hello World" dan memodifikasinya menjadi **Inventory App** dengan bantuan *code generation* Gemini.
- Menjelaskan isi `Dockerfile` menggunakan fitur **"Gemini: Explain this"**.
- Men-deploy aplikasi ke Google Cloud (Cloud Run) menggunakan instruksi langkah-demi-langkah dari Gemini.

🛡️ GRC / Compliance Relevance:
- Mempraktikkan **Least Privilege Access** dengan memberikan role IAM khusus (`cloudaicompanion.user`) hanya kepada user yang membutuhkan asistensi AI.
- Memahami pentingnya validasi manusia terhadap output AI untuk menjaga integritas kode aplikasi.

✅ What I Learned:
- AI bukan cuma buat chat, tapi sangat kuat untuk bantu menulis Dockerfile dan route API.
- Cara kerja emulator Cloud Run untuk testing lokal sebelum benar-benar di-deploy ke cloud.
